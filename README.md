How to run the code

run tests 

```python tests.py```

Usage:

python openTargetsEMBL.py [-t/-d] [searchTerm]

-t     Target association search,
-d     Disease association search,
--help Returns doc string

Examples - 

```python openTargetsEMBL.py -d alpers```
```python openTargetsEMBL.py -t ENSG00000264781```
